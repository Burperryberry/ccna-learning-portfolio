# Flashcard Progress

The [Anki CCNA progress dashboard](<Anki Progress Dashboard.md>) records:

- Current card workload
- Daily review volume and study time
- Seven-day and 30-day momentum
- Review streaks and mature-card retention
- Topics with the highest recent Again rate

The dashboard is a dated snapshot exported from the local CCNA Obsidian vault. It contains learning metrics only—not the flashcard collection itself.
